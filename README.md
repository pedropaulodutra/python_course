# Python Course

Welcome! This repository is part of a personal mission. It's open, structured, and designed to help anyone who wants to learn Python hands-on, no prior experience required.

---

## Who Is This For?

-   Beginners who want to start coding in Python
-   Self-learners looking for structured, example-driven material
-   Anyone interested in open learning

---

## How to Use

Clone the repo:

```bash
git clone https://github.com/pedropaulodutra/python_course.git
cd python_course
```

Navigate through folders, read the `.md` files, and try editing the `.py` files to practice.
